## 🚀 Getting Started

```bash
git clone https://github.com/Krispy145/react-native-chat-rag.git
cd react-native-chat-rag
npm install
```

**Run:**
```bash
npm start
```

**iOS:**
```bash
npx expo run:ios
```

**Android:**
```bash
npx expo run:android
```

**Generate Components:**
```bash
npm run plop
```
